Challenge

This lab involves fetching data from the XKCD API and displaying the result on a web page. The goal is to practice using JSON and APIs in a real-world context.

Problems

Before working with the API, we need to test the endpoints and understand the structure of the returned data. Make sure you have the "JSON Formatter" Chrome extension installed to better view the JSON responses.
